Sophys: Common
==============

Sophys: Common is a collection of `Bluesky`_ and `Ophyd`_ utility code (plans, devices, and other useful stuff) for usage at LNLS/SIRIUS beamlines.

Sophys stands for **S**\ irius **Oph**\ yd and Bluesk\ **y** utilitie\ **s**\ . As an old wise monk once said, nothing beats a cool-looking name.
